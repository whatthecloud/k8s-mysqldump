# k8s-mysqldump
[![Docker Pulls](https://img.shields.io/docker/pulls/whatthecloud/k8s-mysqldump.svg)](https://hub.docker.com/r/whatthecloud/k8s-mysqldump)[![Docker Stars](https://img.shields.io/docker/stars/whatthecloud/k8s-mysqldump.svg)](https://hub.docker.com/r/whatthecloud/k8s-mysqldump)[![License: GPL v3](https://img.shields.io/badge/License-GPLv3-yellow.svg)](https://www.gnu.org/licenses/gpl-3.0)

Based heavily on https://github.com/camilb/kube-mysqldump-cron/ and https://github.com/bigtruedata/docker-mysqldump.

Kubernetes CronJob deployment that can automate backups of MySQL family databases.
Includes optional encryption.
